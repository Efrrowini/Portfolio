# Efrrowini T — Portfolio

Personal portfolio site built as a single-file, no-build HTML/CSS/JS site — showcasing production ML/AI work, live demos, and background.

**Live site:** [https://github.com/Efrrowini/Portfolio/]

## What's inside

- **Real-time Fraud Detection System** — XGBoost + Isolation Forest ensemble, AUC-ROC 0.9833, Redis streaming pipeline, Docker Compose deployment
- **VortexIQ RAG Support Agent** — Conversational RAG agent built during an AI engineering internship, 35/35 eval pass rate, 0% hallucination rate
- **Onboard** — Slack-native volunteer onboarding agent (Slack Bolt + ChromaDB RAG + SQLite), built for the Slack Agent Builder Challenge
- **CodeReview AI** — AI-powered code review tool for students, built solo for the EdTech 3.0 hackathon

Plus certifications, a skills breakdown, and a project timeline.

## Stack

Vanilla HTML/CSS/JS — no framework, no build step. Fonts via Google Fonts, icons via [Tabler Icons](https://tabler.io/icons) and [Devicon](https://devicon.dev/).

## Running locally

No build step required — clone and open `index.html` in a browser:

```bash
git clone https://github.com/Efrrowini/portfolio.git
cd portfolio
open index.html   # or just double-click the file
```
